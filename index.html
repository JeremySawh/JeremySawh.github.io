Pi-Band

What is Pi-band? This is a collection of Raspberry Pi's that allow for musicians to practice their instruments together without being in the same room. Music is streamed through a network in real time to each participants Raspbery Pi and out to their speaker. Practice without compromises.

TFTP Server

This project consited of a client and Server coded in Java that follows the TFTP specifications for file transfers over a network. We were able to demonstrate multiple clients sending and receiving requests from one server in real time and with a test suite we developed as a team.
